React

❤A short, but quite brilliant guide❤

A react is a quite efficiant little Javascript library that helps you create a little interactive UI. For instance, you can easily build a basic version of a "tic, tac, toe"-game that lets the user play the game interactivly by clicking one of the nine boxes of the game.

With the help of little pieces of code, called components, you can build complex UIs. According to the wikipedia page (linked below), it's very good for "fetching rapidly changing data that needs to be recorded".
